OfficeBoost
===========

by OfficeBoost Team

---------------------

老师要求的功能：
1)     公文流转和处理状态跟踪 
2)     会议预约和场地安排 
3)     人员状态查询 
4)     休假审批和状态查询 
5)     设备管理 
6)     邮件提醒、日程、备忘 
7)     报表统计功能 
8)     用户授权管理 

任务列表：
1. JSF(前端表现)
2. EJB(后端接口)
3. JPA(数据持久化)
4. 工作流
5. 组和策略管理
6. 场地和设备管理
7. 人员管理
8. Calendar日程
9. 邮件(只需发送)
10. 报表
11. 文本导入器
